The PropertyHookType Enum

Introduction

(PHP 8 >= 8.4)

The PropertyHookType enum lists the legal types of property hook.

Enum synopsis

enum PropertyHookType
{
case Get ; // Indicates a get hook.

case Set ; // Indicates a set hook.

}
