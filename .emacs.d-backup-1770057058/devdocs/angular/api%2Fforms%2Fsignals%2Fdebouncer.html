Debouncer

A function that defines custom debounce logic for a field.

API

type Debouncer<TValue, TPathKind extends PathKind = PathKind.Root> = (
  context: FieldContext<TValue, TPathKind>,
  abortSignal: AbortSignal,
) => Promise<void> | void
