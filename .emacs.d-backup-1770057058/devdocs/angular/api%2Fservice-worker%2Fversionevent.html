VersionEvent

A union of all event types that can be emitted by SwUpdate#versionUpdates.

API

type VersionEvent = | VersionDetectedEvent
  | VersionInstallationFailedEvent
  | VersionReadyEvent
  | NoNewVersionDetectedEvent
