SimpleChange

Represents a basic change from a previous to a new value for a single property on a directive instance. Passed as a value in a SimpleChanges object to the ngOnChanges hook.

OnChanges

API

class SimpleChange<T = any> {
  constructor(previousValue: T, currentValue: T, firstChange: boolean): SimpleChange<T>;
  override previousValue: T;
  override currentValue: T;
  override firstChange: boolean;
  isFirstChange(): boolean;
}
