AggregateMetadataKey

Represents metadata that is aggregated from multiple parts according to the key's reducer function. A value can be contributed to the aggregated value for a field using an aggregateMetadata rule in the schema. There may be multiple rules in a schema that contribute values to the same AggregateMetadataKey of the same field.
