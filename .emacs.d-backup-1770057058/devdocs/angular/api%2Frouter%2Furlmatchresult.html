UrlMatchResult

Represents the result of matching URLs with a custom matching function.

UrlMatcher

API

type UrlMatchResult = {
  consumed: UrlSegment[];
  posParams?: {[name: string]: UrlSegment};
}

Description

Represents the result of matching URLs with a custom matching function.

  • consumed is an array of the consumed URL segments.
  • posParams is a map of positional parameters.
