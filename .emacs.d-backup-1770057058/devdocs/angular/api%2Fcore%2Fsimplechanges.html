SimpleChanges

A hashtable of changes represented by SimpleChange objects stored at the declared property name they belong to on a Directive or Component. This is the type passed to the ngOnChanges hook. Pass the current class or this as the first generic argument for stronger type checking (e.g. SimpleChanges<YourComponent>).
