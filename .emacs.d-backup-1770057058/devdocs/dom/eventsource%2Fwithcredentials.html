EventSource: withCredentials property

Baseline Widely available

This feature is well established and works across many devices and browser versions. It’s been available across browsers since ⁨January 2020⁩.

  • Learn more
  • See full compatibility
  • Report feedback

Note: This feature is available in Web Workers.

The withCredentials read-only property of the EventSource interface returns a boolean value indicating whether the EventSource object was instantiated with CORS credentials set.

Value

A boolean value indicating whether the EventSource object was instantiated with CORS credentials set (true), or not (false, the default).

Examples

const evtSource = new EventSource("sse.php");
console.log(evtSource.withCredentials);

Note: You can find a full example on GitHub — see Simple SSE demo using PHP.
