TextDecoder: ignoreBOM property

Note: This feature is available in Web Workers.

The ignoreBOM read-only property of the TextDecoder interface is a Boolean indicating whether the byte order mark will be included in the output or skipped over. The value of the property is set in the ignoreBOM argument to the decoder's constructor.

Value

true if the byte order mark will be included in the decoded text; false if it will be skipped over when decoding and omitted.
