FontFace: variant property

Limited availability

This feature is not Baseline because it does not work in some of the most widely-used browsers.

Note: This feature is available in Web Workers.

Non-standard: This feature is not standardized. We do not recommend using non-standard features in production, as they have limited browser support, and may change or be removed. However, they can be a suitable alternative in specific cases where no standard option exists.

The variant property of the FontFace interface programmatically retrieves or sets font variant values.

Value

A string containing a descriptor as it would be defined in a style sheet's @font-face rule.

Specifications

This feature does not appear to be defined in any specification.>
