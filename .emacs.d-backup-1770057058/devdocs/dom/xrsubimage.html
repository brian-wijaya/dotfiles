The XRSubImage interface of the WebXR Device API represents what viewport of the GPU texture to use for rendering.

Interfaces based on XRSubImage

Below is a list of interfaces based on the XRSubImage interface.

  • XRWebGLSubImage: used during rendering of WebGL layers.

Instance properties

XRSubImage.viewport Read only Experimental

The XRViewport used when rendering the sub image.
