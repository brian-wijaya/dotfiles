FileSystemEntry: name property

The read-only name property of the FileSystemEntry interface returns a string specifying the entry's name; this is the entry within its parent directory (the last component of the path as indicated by the fullPath property).

Value

A string indicating the entry's name.

Examples

This example shows a function called isFileWithExtension() which returns true if the specified FileSystemEntry is both a file and the file's name ends with a given extension.

function isFileWithExtension(entry, extension) {
  return entry.isFile && entry.name.endsWith(`.${extension}`);
}
