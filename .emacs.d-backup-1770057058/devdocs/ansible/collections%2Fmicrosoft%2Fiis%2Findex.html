Plugin Index

These are the plugins in the microsoft.iis collection:

Modules

  • virtual_directory module – Configures a virtual directory in IIS
  • virtual_directory_info module – Retrive information on a virtual directory in IIS
  • web_app_pool module – Configure IIS Web Application Pools
  • web_app_pool_info module – Get information on IIS Web Application Pools
  • web_application module – Configures IIS web applications
  • web_application_info module – Get information on IIS Web Applications
  • website module – Configures an IIS website
  • website_info module – Get information on IIS websites
