purestorage.flashblade.purefb_fleet module – Manage Fusion Fleet

Note

This module is part of the purestorage.flashblade collection (version 1.20.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install purestorage.flashblade. You need further requirements to be able to use this module, see Requirements for details.

To use it in a playbook, specify: purestorage.flashblade.purefb_fleet.

New in purestorage.flashblade 1.20.0

Synopsis

  • Create/Modify/Delete Fusion fleet and members

Requirements

The below requirements are needed on the host that executes this module.

  • python >= 3.9
  • py-pure-client
  • purity_fb >= 1.12.2
  • netaddr
  • datetime
  • pytz
  • distro
  • pycountry
  • urllib3

Parameters

Parameter

Comments

api_token

string

FlashBlade API token for admin privileged user.

disable_warnings

boolean

added in purestorage.flashblade 1.18.0

Disable insecure certificate warnings

Choices:

  • false ← (default)
  • true

fb_url

string

FlashBlade management IP address or Hostname.

member_api

string

API token for target array

member_url

string

Management IP address/FQDN of array to add to fleet.

name

string

Name of the fleet.

If not provided local array will be used.

rename

string

new name for fleet

state

string

Define whether to add or remove member from a fleet.

Create a new fleet if one does not exist. This will use the current array as the first member.

Fleet deletion can only occiur when the current array is the only fleet member.

Choices:

  • "absent"
  • "present" ← (default)
  • "create"

Notes

Note

  • This module requires the purity_fb Python library
  • You must set PUREFB_URL and PUREFB_API environment variables if fb_url and api_token arguments are not passed to the module directly

Examples

- name: Create a new fleet
  purestorage.flashblade.purefb_fleet:
    name: foo
    state: create
    fb_url: 10.10.10.2
    api_token: T-55a68eb5-c785-4720-a2ca-8b03903bf641

- name: Add a member to fleet foo
  purestorage.flashblade.purefb_fleet:
    name: foo
    member_url: array2
    member_api: c6033033-fe69-2515-a9e8-966bb7fe4b40
    fb_url: 10.10.10.2
    api_token: T-55a68eb5-c785-4720-a2ca-8b03903bf641

- name: Delete a member from fleet foo
  purestorage.flashblade.purefb_fleet:
    name: foo
    member_url: array2
    member_api: c6033033-fe69-2515-a9e8-966bb7fe4b40
    state: absent
    fb_url: 10.10.10.2
    api_token: T-55a68eb5-c785-4720-a2ca-8b03903bf641

- name: Delete fleet foo
  purestorage.flashblade.purefb_fleet:
    name: foo
    state: absent
    fb_url: 10.10.10.2
    api_token: T-55a68eb5-c785-4720-a2ca-8b03903bf641
