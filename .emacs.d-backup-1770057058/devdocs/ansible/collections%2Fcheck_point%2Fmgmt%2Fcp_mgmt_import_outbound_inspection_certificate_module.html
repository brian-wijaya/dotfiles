check_point.mgmt.cp_mgmt_import_outbound_inspection_certificate module – Import Outbound Inspection certificate for HTTPS inspection.

Note

This module is part of the check_point.mgmt collection (version 6.4.1).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install check_point.mgmt.

To use it in a playbook, specify: check_point.mgmt.cp_mgmt_import_outbound_inspection_certificate.

New in check_point.mgmt 6.0.0

  • Synopsis
  • Parameters
  • Examples
  • Return Values

Synopsis

  • Import Outbound Inspection certificate for HTTPS inspection.
  • All operations are performed over Web Services API.
  • Available from R81.20 management version.
