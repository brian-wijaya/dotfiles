Plugin Index

These are the plugins in the dellemc.enterprise_sonic collection:

Modules

  • sonic_aaa module – Manage AAA and its parameters
  • sonic_acl_interfaces module – Manage access control list (ACL) to interface binding on SONiC
  • sonic_api module – Manages REST operations on devices running Enterprise SONiC
  • sonic_bfd module – Manage BFD configuration on SONiC
  • sonic_bgp module – Manage global BGP and its parameters
  • sonic_bgp_af module – Manage global BGP address-family and its parameters
  • sonic_bgp_as_paths module – Manage BGP autonomous system path (or as-path-list) and its parameters
  • sonic_bgp_communities module – Manage BGP community and its parameters
  • sonic_bgp_ext_communities module – Manage BGP extended community-list and its parameters
  • sonic_bgp_neighbors module – Manage a BGP neighbor and its parameters
  • sonic_bgp_neighbors_af module – Manage the BGP neighbor address-family and its parameters
  • sonic_command module – Runs commands on devices running Enterprise SONiC
  • sonic_config module – Manages configuration sections on devices running Enterprise SONiC
  • sonic_copp module – Manage CoPP configuration on SONiC
  • sonic_dhcp_relay module – Manage DHCP and DHCPv6 relay configurations on SONiC
  • sonic_dhcp_snooping module – Manage DHCP Snooping on SONiC
  • sonic_facts module – Collects facts on devices running Enterprise SONiC
  • sonic_fips module – Manage FIPS configurations on SONiC
  • sonic_image_management module – Manage installation of Enterprise SONiC image, software patch and firmware updater
  • sonic_interfaces module – Configure Interface attributes on interfaces such as, Eth, LAG, VLAN, and loopback. (create a loopback interface if it does not exist.)
  • sonic_ip_neighbor module – Manage IP neighbor global configuration on SONiC.
  • sonic_l2_acls module – Manage Layer 2 access control lists (ACL) configurations on SONiC
  • sonic_l2_interfaces module – Configure interface-to-VLAN association that is based on access or trunk mode
  • sonic_l3_acls module – Manage Layer 3 access control lists (ACL) configurations on SONiC
  • sonic_l3_interfaces module – Configure the IPv4 and IPv6 parameters on Interfaces such as, Eth, LAG, VLAN, and loopback
  • sonic_lag_interfaces module – Manage link aggregation group (LAG) interface parameters
  • sonic_ldap module – Configure global LDAP server settings on SONiC.
  • sonic_lldp_global module – Manage Global LLDP configurations on SONiC
  • sonic_lldp_interfaces module – Manage Inteface LLDP configurations on SONiC
  • sonic_logging module – Manage logging configuration on SONiC.
  • sonic_login_lockout module – Manage Global Login Lockout configurations on SONiC
  • sonic_mac module – Manage MAC configuration on SONiC
  • sonic_mclag module – Manage multi chassis link aggregation groups domain (MCLAG) and its parameters
  • sonic_mgmt_servers module – Manage management servers configuration on SONiC
  • sonic_ntp module – Manage NTP configuration on SONiC.
  • sonic_ospf_area module – configure OSPF area settings on SONiC
  • sonic_ospfv2 module – Configure global OSPFv2 protocol settings on SONiC.
  • sonic_ospfv2_interfaces module – Configure OSPFv2 interface mode protocol settings on SONiC.
  • sonic_pim_global module – Manage global PIM configurations on SONiC
  • sonic_pim_interfaces module – Manage interface-specific PIM configurations on SONiC
  • sonic_pki module – Manages PKI attributes of Enterprise Sonic
  • sonic_poe module – Manage PoE configuration on SONiC
  • sonic_port_breakout module – Configure port breakout settings on physical interfaces
  • sonic_port_group module – Manages port group configuration on SONiC.
  • sonic_prefix_lists module – prefix list configuration handling for SONiC
  • sonic_qos_buffer module – Manage QoS buffer configuration on SONiC
  • sonic_qos_interfaces module – Manage QoS interfaces configuration on SONiC
  • sonic_qos_maps module – Manage QoS maps configuration on SONiC
  • sonic_qos_pfc module – Manage QoS PFC configuration on SONiC
  • sonic_qos_scheduler module – Manage QoS scheduler configuration on SONiC
  • sonic_qos_wred module – Manage QoS WRED profiles configuration on SONiC
  • sonic_radius_server module – Manage RADIUS server and its parameters
  • sonic_roce module – Manage RoCE QoS configuration on SONiC
  • sonic_route_maps module – route map configuration handling for SONiC
  • sonic_sflow module – configure sflow settings on SONiC
  • sonic_static_routes module – Manage static routes configuration on SONiC
  • sonic_stp module – Manage STP configuration on SONiC
  • sonic_system module – Configure system parameters
  • sonic_tacacs_server module – Manage TACACS server and its parameters
  • sonic_users module – Manage users and its parameters
  • sonic_vlan_mapping module – Configure vlan mappings on SONiC.
  • sonic_vlans module – Manage VLAN and its parameters
  • sonic_vrfs module – Manage VRFs and associate VRFs to interfaces such as, Eth, LAG, VLAN, and loopback
  • sonic_vrrp module – Configure VRRP protocol settings on SONiC.
  • sonic_vxlans module – Manage VxLAN EVPN and its parameters

Cliconf Plugins

  • sonic cliconf – Use sonic cliconf to run command on Dell SONiC platforms

Httpapi Plugins

  • sonic httpapi – HttpApi Plugin for devices supporting Restconf SONIC API
