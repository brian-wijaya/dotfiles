Plugin Index

These are the plugins in the dellemc.powerflex collection:

Modules

  • device module – Manage device on Dell PowerFlex
  • fault_set module – Manage Fault Sets on Dell PowerFlex
  • info module – Gathering information about Dell PowerFlex
  • mdm_cluster module – Manage MDM cluster on Dell PowerFlex
  • nvme_host module – Manage NVMe hosts on Dell PowerFlex
  • protection_domain module – Manage Protection Domain on Dell PowerFlex
  • replication_consistency_group module – Manage replication consistency groups on Dell PowerFlex
  • replication_pair module – Manage replication pairs on Dell PowerFlex
  • resource_group module – Manage resource group deployments on Dell PowerFlex.
  • sdc module – Manage SDCs on Dell PowerFlex
  • sds module – Manage SDS on Dell PowerFlex
  • sdt module – Manage SDT (also called NVMe Target) on Dell PowerFlex
  • snapshot module – Manage Snapshots on Dell PowerFlex
  • snapshot_policy module – Manage snapshot policies on Dell PowerFlex
  • storagepool module – Managing Dell PowerFlex storage pool
  • volume module – Manage volumes on Dell PowerFlex

Role Index

These are the roles in the dellemc.powerflex collection:

  • powerflex_activemq role – Role to manage the installation and uninstallation of Powerflex ActiveMQ.
  • powerflex_config role – Role to configure the protection domain, fault set and storage pool.
  • powerflex_gateway role – Role to manage the installation and uninstallation of PowerFlex gateway
  • powerflex_sdr role – Role to manage installation and uninstallation Powerflex SDR
  • powerflex_tb role – Role to manage the installation and uninstallation of Powerflex TB.
  • powerflex_webui role – Role to manage the installation and uninstallation of Powerflex web UI.
