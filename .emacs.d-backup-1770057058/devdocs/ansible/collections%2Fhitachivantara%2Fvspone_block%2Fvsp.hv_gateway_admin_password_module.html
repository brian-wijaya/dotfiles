Alternative:

Synopsis

  • This module updates the password of gateway admin.
  • For examples go to URL https://github.com/hitachi-vantara/vspone-block-ansible/blob/main/playbooks/vsp_uai_gateway/admin_password.yml

Requirements

The below requirements are needed on the host that executes this module.

  • python >= 3.9

Parameters

Parameter

Comments

connection_info

dictionary / required

Information required to establish a connection to the storage system.

api_token

string / required

Token value to access UAI gateway. This is a required field for gateway connection type.

uai_gateway_address

string / required

IP address or hostname of UAI gateway.

spec

dictionary

Specification for updating admin password.

password

string / required

The new password value to be updated.

Examples

- name: Update password of UAI gateway admin
  hitachivantara.vspone_block.vsp.hv_gateway_admin_password:
    connection_info:
      uai_gateway_address: gateway.company.com
      api_token: "eyJhbGciOiJS......"
    spec:
      password: "changeMe!"
