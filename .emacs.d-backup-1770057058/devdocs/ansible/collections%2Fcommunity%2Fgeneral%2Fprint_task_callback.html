Callback plugin

This plugin is an aggregate callback. It adds additional console output next to the configured stdout callback. See Callback plugins for more information on callback plugins.

Synopsis

  • This plugin prints the currently executing playbook task to the job output.

Requirements

The below requirements are needed on the local controller node that executes this callback.

  • enable in configuration

Examples

ansible.cfg: >
    # Enable plugin
    [defaults]
    callbacks_enabled=community.general.print_task
