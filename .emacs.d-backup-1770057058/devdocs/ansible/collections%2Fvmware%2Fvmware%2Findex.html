Plugin Index

These are the plugins in the vmware.vmware collection:

Modules

  • appliance_info module – Gather appliance information
  • cluster module – Manage VMware vSphere clusters
  • cluster_dpm module – Manage Distributed Power Management (DPM) on VMware vSphere clusters
  • cluster_drs module – Manage Distributed Resource Scheduler (DRS) on VMware vSphere clusters
  • cluster_drs_recommendations module – Apply Distributed Resource Scheduler (DRS) recommendations on VMware vSphere clusters
  • cluster_ha module – Manage High Availability services (HA) on VMware vSphere clusters
  • cluster_info module – Gathers information about one or more clusters
  • cluster_vcls module – Manage the vCLS (vSphere Cluster Services) VM disk placement for this cluster.
  • content_library_item_info module – Gather info about content library items.
  • content_template module – Manage template in content library from virtual machine.
  • deploy_content_library_ovf module – Deploy a virtual machine from an OVF in a content library.
  • deploy_content_library_template module – Deploy a virtual machine from a template in a content library.
  • deploy_folder_template module – Deploy a VM from a template located in a folder
  • esxi_connection module – Manage VMware ESXi host connection status in vCenter
  • esxi_host module – Manage VMware ESXi host status in vCenter
  • esxi_maintenance_mode module – Manage an ESXi hosts maintenance mode setting in vCenter
  • folder module – Manage VMware vSphere folders
  • folder_template_from_vm module – Create a template in a local VCenter folder from an existing VM
  • guest_info module – Gather guest information
  • import_content_library_ovf module – Import an OVF or OVA file to a content library from a remote source.
  • license_info module – Fetch VMware vCenter license keys
  • local_content_library module – Manage a local content library.
  • subscribed_content_library module – Manage a subscribed content library.
  • vcsa_settings module – Configure vCenter Server Appliance settings
  • vm_list_group_by_clusters_info module – Returns information about the virtual machines grouping by clusters and folders
  • vm_portgroup_info module – Returns information about the portgroups of virtual machines
  • vm_powerstate module – Manages power states of virtual machines in vCenter
  • vm_resource_info module – Gather information about the resources of one or more VMs.

Inventory Plugins

  • esxi_hosts inventory – Create an inventory containing VMware ESXi hosts
  • vms inventory – Create an inventory containing VMware VMs
