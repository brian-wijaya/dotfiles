headtooltip

Tooltip annotation attached to the head of an edge

type: escString, default: ""

Used only if the edge has a headURL attribute.

See also:

  • edgetooltip.
  • labeltooltip.
  • tailtooltip.
  • tooltip.
Valid on:
  • Edges

Note: cmap, svg only.
