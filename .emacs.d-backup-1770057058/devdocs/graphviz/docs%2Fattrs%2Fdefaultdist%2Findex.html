defaultdist

The distance between nodes in separate connected components

type: double, default: 1+(avg. len)*sqrt(|V|), minimum: epsilon

If set too small, connected components may overlap.

Only applicable if pack=false.

Valid on:
  • Graphs

Note: neato only.
