Kitty

Kitty graphics protocol output

Since Graphviz 9.0.0, an output format for speaking the Kitty terminal emulator’s graphics protocol is available.

  • kitty: uncompressed Kitty graphics protocol output
  • kittyz: compressed Kitty graphics protocol output

These formats assume your terminal is either Kitty or something that understands Kitty’s protocol.
