arrowsize

Multiplicative scale factor for arrowheads

type: double, default: 1.0, minimum: 0.0

Example 
digraph {
  quiver -> "0.5" [arrowsize=0.5]
  quiver -> "1"
  quiver -> "2" [arrowsize=2]
  quiver -> "3" [arrowsize=3]
}
