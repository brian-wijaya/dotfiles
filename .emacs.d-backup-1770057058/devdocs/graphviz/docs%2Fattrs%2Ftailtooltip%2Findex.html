tailtooltip

Tooltip annotation attached to the tail of an edge

type: escString, default: ""

Used only if the edge has a tailURL attribute.

  • edgetooltip.
  • headtooltip.
  • labeltooltip.
  • tooltip.
Valid on:
  • Edges

Note: cmap, svg only.
