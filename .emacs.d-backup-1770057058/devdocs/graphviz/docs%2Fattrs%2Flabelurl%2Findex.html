labelURL

If defined, labelURL is the link used for the label of an edge

type: escString, default: ""

labelURL overrides any URL defined for the edge.

See also:

  • edgehref, edgeURL
  • headhref, headURL
  • labelhref, labelURL
  • tailhref, tailURL
  • href, URL
Valid on:
  • Edges

Note: map, svg only.
