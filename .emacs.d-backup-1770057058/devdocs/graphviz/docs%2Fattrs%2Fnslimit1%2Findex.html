nslimit1

Sets number of iterations in network simplex applications

type: double

nslimit1 is used for ranking nodes.

If defined, # iterations = nslimit * # nodes; otherwise, # iterations = INT_MAX.

Valid on:
  • Graphs

Note: dot only.
