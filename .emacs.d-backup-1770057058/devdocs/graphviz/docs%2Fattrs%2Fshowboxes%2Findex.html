showboxes

Print guide boxes for debugging

type: int, default: 0, minimum: 0

Print guide boxes in PostScript at the beginning of routesplines if showboxes=1, or at the end if showboxes=2. (Debugging, TB mode only!)

Valid on:
  • Edges
  • Nodes
  • Graphs

Note: dot only.
