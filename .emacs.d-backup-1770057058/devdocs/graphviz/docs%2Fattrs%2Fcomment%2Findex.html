comment

Comments are inserted into output

type: string, default: ""

Device-dependent.

Example 
digraph {
  comment="I am a graph"
  A [comment="I am node A"]
  B [comment="I am node B"]
  A->B [comment="I am an edge"]
}
Edit in Playground
