Add a development dependency

To add an npm package as a development dependency, use bun add --development. 
bun add zod --dev
bun add zod -d # shorthand
This will add the package to devDependencies in package.json. 
{
  "devDependencies": {
    "zod": "^3.0.0"
  }
}
