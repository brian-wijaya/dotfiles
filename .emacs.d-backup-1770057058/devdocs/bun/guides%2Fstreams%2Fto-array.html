Convert a ReadableStream to an array of chunks

Bun provides a number of convenience functions for reading the contents of a ReadableStream into different formats. The Bun.readableStreamToArray function reads the contents of a ReadableStream to an array of chunks. 
const stream = new ReadableStream();
const str = await Bun.readableStreamToArray(stream);
