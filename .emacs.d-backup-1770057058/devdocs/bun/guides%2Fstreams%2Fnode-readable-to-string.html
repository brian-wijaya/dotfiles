Convert a Node.js Readable to a string

To convert a Node.js Readable stream to a string in Bun, you can create a new Response object with the stream as the body, then use response.text() to read the stream into a string. 
import { Readable } from "stream";
const stream = Readable.from([Buffer.from("Hello, world!")]);
const text = await new Response(stream).text();
console.log(text); // "Hello, world!"
