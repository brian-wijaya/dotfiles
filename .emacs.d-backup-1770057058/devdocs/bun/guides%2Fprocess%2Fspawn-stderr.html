Read stderr from a child process

When using Bun.spawn(), the child process inherits the stderr of the spawning process. If instead youâ€™d prefer to read and handle stderr, set the stderr option to "pipe". 
const proc = Bun.spawn(["echo", "hello"], {
  stderr: "pipe",
});

proc.stderr; // => ReadableStream
To read stderr until the child process exits, use .text() 
const proc = Bun.spawn(["echo", "hello"], {
  stderr: "pipe",
});

const errors: string = await proc.stderr.text();
if (errors) {
  // handle errors
}
