Get the current Bun version

Get the current version of Bun in a semver format. 
Bun.version; // => "1.3.3"
Get the exact git commit of oven-sh/bun that was compiled to produce this Bun binary. 
Bun.revision; // => "49231b2cb9aa48497ab966fc0bb6b742dacc4994"
See Docs > API > Utils for more useful utilities.
