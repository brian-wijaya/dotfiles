Convert a ReadableStream to a Buffer

Bun provides a number of convenience functions for reading the contents of a ReadableStream into different formats. This snippet reads the contents of a ReadableStream to an ArrayBuffer, then creates a Buffer that points to it. 
const stream = new ReadableStream();
const arrBuf = await Bun.readableStreamToArrayBuffer(stream);
const nodeBuf = Buffer.from(arrBuf);
