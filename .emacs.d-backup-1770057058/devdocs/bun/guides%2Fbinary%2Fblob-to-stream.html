Convert a Blob to a ReadableStream

The Blob class provides a number of methods for consuming its contents in different formats, including .stream(). This returns Promise<ReadableStream>. 
const blob = new Blob(["hello world"]);
const stream = await blob.stream();
See Docs > API > Binary Data for complete documentation on manipulating binary data with Bun.
