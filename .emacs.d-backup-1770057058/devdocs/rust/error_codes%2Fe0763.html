Error code E0763

A byte constant wasn't correctly ended.

Erroneous code example:

#![allow(unused)]
fn main() {
let c = b'a; // error!
}

To fix this error, add the missing quote:

#![allow(unused)]
fn main() {
let c = b'a'; // ok!
}
