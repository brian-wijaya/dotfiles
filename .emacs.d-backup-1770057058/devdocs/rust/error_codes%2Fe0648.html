Error code E0648

An export_name attribute contains null characters (\0).

Erroneous code example:

#![allow(unused)]
fn main() {
#[export_name="\0foo"] // error: `export_name` may not contain null characters
pub fn bar() {}
}

To fix this error, remove the null characters:

#![allow(unused)]
fn main() {
#[export_name="foo"] // ok!
pub fn bar() {}
}
