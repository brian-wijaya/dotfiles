Error code E0765

A double quote string (") was not terminated.

Erroneous code example:

#![allow(unused)]
fn main() {
let s = "; // error!
}

To fix this error, add the missing double quote at the end of the string:

#![allow(unused)]
fn main() {
let s = ""; // ok!
}
