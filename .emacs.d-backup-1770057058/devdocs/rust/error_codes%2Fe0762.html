Error code E0762

A character literal wasn't ended with a quote.

Erroneous code example:

#![allow(unused)]
fn main() {
static C: char = '●; // error!
}

To fix this error, add the missing quote:

#![allow(unused)]
fn main() {
static C: char = '●'; // ok!
}
