Error code E0766

A double quote byte string (b") was not terminated.

Erroneous code example:

#![allow(unused)]
fn main() {
let s = b"; // error!
}

To fix this error, add the missing double quote at the end of the string:

#![allow(unused)]
fn main() {
let s = b""; // ok!
}
