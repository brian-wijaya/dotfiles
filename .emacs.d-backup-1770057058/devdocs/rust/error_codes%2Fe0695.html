Error code E0695

A break statement without a label appeared inside a labeled block.

Erroneous code example:

#![allow(unused)]
fn main() {
loop {
    'a: {
        break;
    }
}
}

Make sure to always label the break:

#![allow(unused)]
fn main() {
'l: loop {
    'a: {
        break 'l;
    }
}
}

Or if you want to break the labeled block:

#![allow(unused)]
fn main() {
loop {
    'a: {
        break 'a;
    }
    break;
}
}
