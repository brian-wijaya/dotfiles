Error code E0768

A number in a non-decimal base has no digits.

Erroneous code example:

#![allow(unused)]
fn main() {
let s: i32 = 0b; // error!
}

To fix this error, add the missing digits:

#![allow(unused)]
fn main() {
let s: i32 = 0b1; // ok!
}
