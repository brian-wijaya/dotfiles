Error code E0770

The type of a const parameter references other generic parameters.

Erroneous code example:

#![allow(unused)]
fn main() {
fn foo<T, const N: T>() {} // error!
}

To fix this error, use a concrete type for the const parameter:

#![allow(unused)]
fn main() {
fn foo<T, const N: usize>() {}
}
