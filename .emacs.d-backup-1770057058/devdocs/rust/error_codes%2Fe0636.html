Error code E0636

The same feature is enabled multiple times with #![feature] attributes

Erroneous code example:

#![allow(unused)]
#![allow(stable_features)]
#![feature(rust1)]
#![feature(rust1)] // error: the feature `rust1` has already been enabled
fn main() {
}
